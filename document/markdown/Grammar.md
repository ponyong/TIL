### MarkDown 기본문법

0. TEXT 박스 ( 주석 개념 )

```
    `(그레이브) 세개로 TEXT 를 감싸주게 되면 텍스트 박스 형태로 저장됩니다.
```

1. HEADER (머리말)

```
# 의 개수로 글씨의 크기를 조절 할 수 있습니다.
1~6개 까지 지원하며 여러개를 사용 할 수록 크기가 작아집니다.
```

# TEST is H1

## TEST is H2

### TEST is H3

2. 강조 ( Emphasis )

```
각각 <em> <strong> <del> 태그로 변환됩니다.

밑줄은 <u></u> 태그를 사용합니다.
```

```
이텔릭체는 * 혹은 _ 를 사용합니다.
ex) * Example *

볼드체는 ** 혹은 __ 를 사용합니다.
ex) ** Example **

두가지를 혼합하여 사용 할 수 있습니다.
ex) **_ Example _**

취소선은 물결표시를 사용합니다.
ex) ~~ Example ~~

밑불은 <u></u> 를 사용합니다.
ex) <u> Example </u>
```

_이텔릭체_ <br>
**볼드체** <br>
**_혼합_** <br>
~~취소선~~ <br>
<u>밑줄</u> <br>
