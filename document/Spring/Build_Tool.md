# Build Tool ( 빌드 도구 )

### 빌드 도구란 ?
+ 소스 코드의 컴파일 , 테스트 , 정적분석 등을 통해 실행 가능한 애플리케이션으로 자동 생성하는 프로그램
+ 라이브러리의 자동 추가 및 관리
+ 라이브러리의 버전을 자동으로 동기화

### 탄생 배경

```
웹, 앱 프로그래밍의 개발이 발전하면서 개발에 필요한 라이브러리의 수도 많아지게 되었는데
많은 라이브러리들을 직접 다운로드하여 추가하는 방법은 너무 비효율적이기에 
효율적으로 개선하기 위해 발생한 것이 빌드 도구이다.
```

### 빌드 도구의 종류 

+ Ant
+ Maven
+ Gradle
